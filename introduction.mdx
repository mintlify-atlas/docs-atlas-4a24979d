## What is SaludMap?

SaludMap is a comprehensive healthcare facility locator platform that helps users find and connect with healthcare providers in their area. Built with React and NestJS, it combines real-time map-based search, appointment booking, user reviews, and offline capabilities into a seamless experience.

## Key Features

<AccordionGroup>
  <Accordion title="Interactive Map Search" icon="map">
    Real-time search for healthcare facilities using Leaflet maps with geolocation support, custom filters by facility type, and auto-centering on your current location.
  </Accordion>
  
  <Accordion title="Appointment System" icon="calendar">
    Complete appointment booking system (turnos) with specialty selection, date/time scheduling, and appointment management for both users and establishments.
  </Accordion>
  
  <Accordion title="Reviews & Ratings" icon="star-half-stroke">
    User-generated reviews and ratings for healthcare facilities, helping others make informed decisions about their healthcare providers.
  </Accordion>
  
  <Accordion title="Offline Support" icon="download">
    Precache map tiles and facility data for specific areas, enabling full offline functionality with automatic synchronization when back online.
  </Accordion>
  
  <Accordion title="AI Chatbot Assistant" icon="robot">
    Powered by Google Gemini, the integrated chatbot helps users find facilities, answer questions, and navigate the platform.
  </Accordion>
  
  <Accordion title="Multi-Language Support" icon="language">
    Full internationalization (i18n) with support for multiple languages, making the platform accessible to diverse communities.
  </Accordion>
</AccordionGroup>

## Technology Stack

SaludMap is built with modern, scalable technologies:

<CardGroup cols={3}>
  <Card title="Frontend" icon="react">
    React 19 with Vite, Leaflet maps, React Router, and i18next for internationalization
  </Card>
  <Card title="Backend" icon="server">
    NestJS with Prisma ORM, JWT authentication, and Google Gemini integration
  </Card>
  <Card title="Database" icon="database">
    MySQL with Prisma for type-safe database access and migrations
  </Card>
</CardGroup>
