## Overview

The SaludMap backend is built with NestJS 11.1.6, following a modular architecture with TypeScript. It uses Prisma as the ORM, JWT for authentication, and integrates Google Gemini AI for chatbot functionality.

**Location**: `backend/src/`

## Main Application

### main.ts

Application bootstrap file:
- Creates NestJS application
- Configures global prefix (`/api`)
- Sets up CORS
- Starts server on port 3000 (default)
- Handles graceful shutdown

```typescript
async function bootstrap() {
  const app = await NestFactory.create(AppModule);
  app.setGlobalPrefix('api');
  app.enableCors({
    origin: '*',
    methods: 'GET,POST,PUT,DELETE',
    credentials: true,
  });
  await app.listen(process.env.PORT || 3000);
}
```

**Location**: `main.ts:1`

## AI Chatbot System

### Chatbot Controller

**Location**: `controllers/chatbot.controller.ts`

Chatbot endpoints:
- `POST /chatbot/message` - Send message to chatbot
- Integrates with Gemini AI service
- Uses Model Context Protocol (MCP)

```typescript
@Post('message')
async sendMessage(@Body() messageDto: ChatMessageDto) {
  return await this.geminiService.processMessage(messageDto);
}
```

**Location**: `controllers/chatbot.controller.ts:1`

**Location**: `services/gemini.service.ts`

## Common Patterns

### Service with Prisma

```typescript
import { Injectable } from '@nestjs/common';
import { PrismaClient } from '../generated';

@Injectable()
export class MyService {
  private prisma = new PrismaClient();

  async findAll() {
    return await this.prisma.myModel.findMany();
  }
}
```

### Protected Endpoint

```typescript
import { Controller, Get, UseGuards } from '@nestjs/common';
import { JwtAuthGuard } from '../auth/jwt-auth.guard';

@Controller('protected')
export class ProtectedController {
  @UseGuards(JwtAuthGuard)
  @Get()
  getProtectedData() {
    return { message: 'This is protected' };
  }
}
```

### DTO Validation

```typescript
import { IsString, IsEmail, MinLength } from 'class-validator';

export class CreateUserDto {
  @IsString()
  nombre: string;

  @IsEmail()
  mail: string;

  @IsString()
  @MinLength(6)
  contrasenia: string;
}
```

## Error Handling

### HTTP Exceptions

```typescript
import { HttpException, HttpStatus } from '@nestjs/common';

if (!user) {
  throw new HttpException('User not found', HttpStatus.NOT_FOUND);
}
```

### Try-Catch Pattern

```typescript
async create(data: CreateDto) {
  try {
    return await this.prisma.model.create({ data });
  } catch (error) {
    throw new HttpException(
      'Failed to create record',
      HttpStatus.INTERNAL_SERVER_ERROR
    );
  }
}
```

## Security Best Practices

1. **Password Hashing**: Always hash passwords with bcrypt
2. **JWT Secrets**: Use strong, random secrets in production
3. **CORS**: Restrict origins in production
4. **Validation**: Use DTOs with class-validator
5. **Guards**: Protect sensitive endpoints
6. **Environment Variables**: Never commit `.env` files
