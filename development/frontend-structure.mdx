## Overview

The SaludMap frontend is built with React 19.1.1, Vite 7.1.7, and uses modern React patterns including hooks, context, and functional components. The codebase is organized by feature with shared utilities and services.

**Location**: `Frontend/src/`

## Core Components

### App.jsx

Main application component that manages:
- Navigation state
- Theme switching (dark/light mode)
- User authentication state
- Tab routing (Map, Appointments, Insurance)
- Mobile menu handling

**Key Features**:
- Tab-based navigation system
- Responsive navbar with mobile hamburger menu
- Theme persistence in localStorage
- Location tracking integration

```jsx
// Main sections rendered based on activeTab
const renderActiveSection = () => {
  switch (activeTab) {
    case 'mapa':      return <MapComponent />;
    case 'analytics': return <Analytics />;
    case 'turnos':    return <Turnos />;
    case 'seguros':   return <InsuranceSection />;
  }
};
```

**Location**: `App.jsx:1`

## Services Layer

### locationService.js

Geolocation management service:

```javascript
class LocationService {
  getCurrentPosition()     // Get current GPS location
  subscribe(callback)      // Listen to location changes
  updateLocation(lat, lng) // Manually set location
  loadLastKnownLocation()  // Load from localStorage
}
```

**Features**:
- Real-time location tracking
- Observable pattern for location updates
- Manual location override
- Persistent location storage
- GPS vs manual source tracking

**Location**: `services/locationService.js:1`

---

**Features**:

## Routing

### Tab-Based Navigation

Instead of React Router for main sections, SaludMap uses a tab system:

```jsx
const [activeTab, setActiveTab] = useState('mapa');
```

Tabs:
- `mapa` - Map view
- `turnos` - Appointments
- `seguros` - Insurance
- `analytics` - Analytics (when establishment selected)

### Custom Events

For inter-component communication:

```javascript
// Trigger tab change
window.dispatchEvent(new CustomEvent('saludmap:change-tab', { 
  detail: { tab: 'turnos' } 
}));

// Trigger map action
window.dispatchEvent(new CustomEvent('map-action', { 
  detail: 'calibrate' 
}));
```

## Common Patterns

### API Calls

```jsx
import axios from 'axios';
import { API_BASE_URL } from '../config/api';

const fetchData = async () => {
  try {
    const response = await axios.get(`${API_BASE_URL}/endpoint`);
    return response.data;
  } catch (error) {
    console.error('API Error:', error);
    throw error;
  }
};
```

### Authentication Headers

```javascript
const token = localStorage.getItem('token');
const config = {
  headers: { Authorization: `Bearer ${token}` }
};
await axios.get(`${API_BASE_URL}/protected`, config);
```

### Toast Notifications

```jsx
import { toast } from 'sonner';

toast.success('Operation successful!');
toast.error('Something went wrong');
toast.info('Information message');
```
