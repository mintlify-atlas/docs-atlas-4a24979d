# SaludMap Documentation

This repository contains the official documentation for SaludMap - a healthcare facility locator with real-time map search, appointment booking, reviews, and offline support.

## About SaludMap

SaludMap is a comprehensive healthcare facility locator platform built with React and NestJS. It helps users find and connect with healthcare providers through:

- Interactive map-based search for healthcare facilities
- Real-time appointment booking system (turnos)
- User reviews and ratings for establishments
- Offline mode with tile and data caching
- AI-powered chatbot assistant (Google Gemini)
- Multi-language support (i18n)

## Local Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview documentation changes locally:

```bash
npm i -g mint
```

Run the following command at the root of the documentation:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.
