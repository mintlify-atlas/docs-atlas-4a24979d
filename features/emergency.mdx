## Overview

The Emergency Widget provides immediate access to critical emergency services. It automatically detects the user's location and displays relevant emergency contact numbers for ambulance and police services.

## Location Detection

### Timezone-Based Detection

The widget uses timezone information to determine the user's country:

```jsx
const [emergencyData, setEmergencyData] = useState({
  countryKey: 'country_international',
  ambulance: '112',
  police: '112'
});

useEffect(() => {
  const timeZone = Intl.DateTimeFormat().resolvedOptions().timeZone || '';

  // Argentina detection
  if (timeZone.includes('Argentina') || 
      timeZone.includes('Buenos_Aires') || 
      timeZone.includes('Mendoza') || 
      timeZone.includes('Cordoba')) {
    setEmergencyData({ 
      countryKey: 'country_argentina', 
      ambulance: '107', 
      police: '911' 
    });
  }
  // France detection
  else if (timeZone.includes('Paris') || timeZone.includes('Europe/')) {
    setEmergencyData({ 
      countryKey: 'country_france', 
      ambulance: '15', 
      police: '17' 
    });
  }
  // Americas detection
  else if (timeZone.includes('America/')) {
    setEmergencyData({ 
      countryKey: 'country_america', 
      ambulance: '911', 
      police: '911' 
    });
  }
}, []);
```

<Info>
The default fallback is the international emergency number 112, which works in most countries.
</Info>

### Supported Regions

| Region | Ambulance | Police | Detection Method |
|--------|-----------|--------|------------------|
| Argentina | 107 | 911 | Timezone: `America/Argentina/*` |
| France | 15 | 17 | Timezone: `Europe/Paris` |
| Americas | 911 | 911 | Timezone: `America/*` |
| International | 112 | 112 | Default fallback |

## Mobile Optimization

### Tel Links

The `tel:` protocol works seamlessly on mobile devices:

```jsx
<a href={`tel:${emergencyData.ambulance}`}>
  {/* Button content */}
</a>
```

On mobile:
- iOS: Opens Phone app with number pre-filled
- Android: Initiates call directly or shows dialer

On desktop:
- Opens default phone/calling app if available
- Otherwise, displays the number

## Event Integration

### Tab Navigation

The "Find Hospital" button triggers navigation:

```jsx
window.dispatchEvent(new CustomEvent('saludmap:change-tab', { 
  detail: { tab: 'mapa' } 
}));
```

This event is caught by the main app component:

```jsx App.jsx
useEffect(() => {
  const handleTabChange = (e) => {
    setActiveTab(e.detail.tab);
  };
  
  window.addEventListener('saludmap:change-tab', handleTabChange);
  return () => window.removeEventListener('saludmap:change-tab', handleTabChange);
}, []);
```

### Focus Management

Modal traps focus when open:

```jsx
useEffect(() => {
  if (isOpen) {
    // Save previously focused element
    const previouslyFocused = document.activeElement;
    
    // Focus first button in modal
    const firstButton = document.querySelector('.emergency-modal button');
    firstButton?.focus();
    
    return () => {
      // Restore focus on close
      previouslyFocused?.focus();
    };
  }
}, [isOpen]);
```

## Testing

<Warning>
