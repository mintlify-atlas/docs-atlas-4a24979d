## Overview

AURA (Asistente Útil de Respuesta Automatizada) is SaludMap's AI-powered chatbot assistant that helps users navigate the platform, answer questions about healthcare facilities, and provide guidance on using features.

## Message Handling

### Sending Messages

User messages are added to the chat and sent to the AI:

```jsx
const handleSend = async (e) => {
  e.preventDefault();
  if (input.trim().length === 0 || isLoading) return;

  // Add user message
  const userMessage = { text: input, sender: 'user' };
  setMessages((prev) => [...prev, userMessage]);
  setInput('');
  setIsLoading(true);

  try {
    // Get AI response
    const auraResponseText = await sendMessageToAura(input);

    // Add AURA's response
    const auraMessage = { text: auraResponseText, sender: 'aura' };
    setMessages((prev) => [...prev, auraMessage]);
  } catch (error) {
    toast.error('Error al obtener respuesta de AURA');
  } finally {
    setIsLoading(false);
  }
};
```

### Auto-Scroll

The chat automatically scrolls to the latest message:

```jsx
const messagesEndRef = useRef(null);

const scrollToBottom = () => {
  messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
};

useEffect(scrollToBottom, [messages]);
```

## Backend Integration

### Chatbot Service

The service sends messages to the backend API:

```jsx chatbotService.js
import axios from 'axios';

export const sendMessageToAura = async (userMessage) => {
  try {
    const response = await axios.post('/api/chatbot', {
      message: userMessage
    });
    
    return response.data.reply || 'Lo siento, no pude procesar tu solicitud.';
  } catch (error) {
    console.error('Error comunicándose con AURA:', error);
    throw error;
  }
};
```

### Google Gemini Integration

The backend uses Google's Gemini AI model:

```javascript server.js (Backend)
const { GoogleGenerativeAI } = require('@google/generative-ai');

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
const model = genAI.getGenerativeModel({ model: 'gemini-pro' });

app.post('/api/chatbot', async (req, res) => {
  try {
    const { message } = req.body;
    
    // System context for healthcare focus
    const context = `You are AURA, a helpful assistant for SaludMap, 
    a healthcare facility locator app. Help users find medical facilities, 
    book appointments, and navigate the platform. Respond in Spanish.`;
    
    const prompt = `${context}\n\nUser: ${message}\nAURA:`;
    
    const result = await model.generateContent(prompt);
    const response = await result.response;
    const text = response.text();
    
    res.json({ reply: text });
  } catch (error) {
    console.error('Gemini API error:', error);
    res.status(500).json({ error: 'Error generating response' });
  }
});
```

<Info>
The backend requires a valid Google Gemini API key in the environment variables.
</Info>

## Chat Features

### Context Awareness

AURA maintains conversation context throughout the session:

```jsx
const [messages, setMessages] = useState([]);

// Messages are preserved until the widget is closed
// This allows AURA to reference previous conversation
```

<Note>
Chat history is cleared when the widget is closed. For persistent chat history, implement session storage or database persistence.
</Note>

### Error Handling

Graceful error handling with user feedback:

```jsx
try {
  const auraResponseText = await sendMessageToAura(input);
  setMessages((prev) => [...prev, { text: auraResponseText, sender: 'aura' }]);
} catch (error) {
  toast.error('Error al obtener respuesta de AURA');
  
  // Add fallback message
  setMessages((prev) => [...prev, {
    text: 'Disculpa, estoy teniendo problemas técnicos. Por favor intenta de nuevo.',
    sender: 'aura'
  }]);
}
```

## Accessibility

### Keyboard Navigation

The chatbot supports keyboard shortcuts:

```jsx
<input
  type="text"
  value={input}
  onChange={(e) => setInput(e.target.value)}
  onKeyDown={(e) => {
    if (e.key === 'Enter' && !e.shiftKey) {
      e.preventDefault();
      handleSend(e);
    }
  }}
  aria-label="Escribe tu mensaje"
/>
```
