## Overview

The appointment booking system ("Turnos" in Spanish) allows authenticated users to request appointments at healthcare facilities. It features specialty search, visual calendar selection, and email confirmation.

## Schedule Parser

### Parsing Opening Hours

The system parses natural language schedule strings from the database:

```jsx
const parsearHorarios = (horariosStr) => {
  // Example input: "Lunes a Viernes: 09:00 a 17:00 | Sábados: 09:00 a 13:00"
  const diasPermitidos = new Set();
  const franjas = [];
  
  const bloques = horariosStr.split(/[|\/]/);
  
  for (const bloque of bloques) {
    // Extract time range: "09:00 a 17:00"
    const horas = [...bloque.matchAll(/\d{1,2}:\d{2}/g)];
    const desde = horas[0];
    const hasta = horas[1];
    
    // Parse day names: "Lunes a Viernes"
    const diasBloque = extractDaysFromText(bloque);
    
    franjas.push({ dias: diasBloque, desde, hasta });
  }
  
  return { diasPermitidos, franjas };
};
```

<Info>
The parser supports Spanish day names with and without accents, ranges ("Lunes a Viernes"), and multiple time blocks.
</Info>

### Generating Time Slots

Once a date is selected, 15-minute time slots are generated based on the specialty's hours:

```jsx
const generarSlots = (franjas, fecha) => {
  const diaSemana = fecha.getDay();
  const slots = [];
  
  for (const franja of franjas) {
    if (!franja.dias.has(diaSemana)) continue;
    
    const [hI, mI] = franja.desde.split(':').map(Number);
    const [hF, mF] = franja.hasta.split(':').map(Number);
    
    let min = hI * 60 + mI;
    const finMin = hF * 60 + mF;
    
    while (min < finMin) {
      slots.push(
        `${String(Math.floor(min / 60)).padStart(2, '0')}:${String(min % 60).padStart(2, '0')}`
      );
      min += 15; // 15-minute intervals
    }
  }
  
  return slots;
};
```

## Saving Appointments

### Backend Integration

Appointments are saved to the database with all relevant details:

```jsx
const handleSolicitarTurno = async (datos) => {
  const turnoGuardado = await guardarTurno({
    usuarioId: user.id,
    establecimientoId: datos.establecimientoId,
    especialidadId: datos.especialidadId,
    fecha: datos.fecha,
    hora: datos.hora,
    observaciones: datos.observaciones

  // Send confirmation email
  await sendAppointmentEmail(
    selected,
    `${datos.fecha}T${datos.hora}`,
    datos.observaciones,
    user.mail,
    `${user.nombre} ${user.apellido}`,
    selectedType,
    prettyType,
    datos.especialidadNombre
```

### Email Confirmation

Confirmation emails are sent using EmailJS:

```jsx emailService.js
export const sendAppointmentEmail = async (
  professional,
  datetime,
  notes,
  userEmail,
  userName,
  professionalType,
  prettyType,
  especialidadNombre
) => {
  const templateParams = {
    to_email: userEmail,
    user_name: userName,
    professional_name: professional.name,
    professional_type: prettyType(professionalType),
    especialidad: especialidadNombre || 'No especificada',
    date: new Date(datetime).toLocaleDateString('es-AR'),
    time: datetime.split('T')[1],
    notes: notes || 'Sin observaciones',
  };
  
  return emailjs.send(SERVICE_ID, TEMPLATE_ID, templateParams);
};
```

```jsx
const handleCancelarTurno = async (id) => {
  toast('¿Estás seguro que querés cancelar este turno?', {
    duration: 8000,
    action: {
      label: 'Sí, cancelar',
      onClick: () => cancelarTurno(id),
    },
    cancel: { label: 'No' },
  });
};

const cancelarTurno = async (turnoId) => {
  const res = await cancelAppointment(turnoId);
  setMisTurnos(prev => prev.filter(t => t.id !== turnoId));
  toast.success('Turno cancelado exitosamente');
};
```

## Integration with Map

### Booking from Map Markers

Users can initiate appointment booking directly from facility markers on the map:

```jsx
// Event listener in Turnos component
useEffect(() => {
  const handleChangeTab = (e) => {
    if (e.detail?.tab !== 'turnos') return;
    
    const { establecimiento, place } = e.detail;
    setPreSelectedEstablecimiento(establecimiento);
    setPreSelectedPlace(place);
    
    setTimeout(() => openModal(place, establecimiento), 100);
  };
  
  window.addEventListener('saludmap:change-tab', handleChangeTab);
}, []);
```
