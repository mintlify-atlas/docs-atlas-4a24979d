---
title: 'List Specialties'
api: 'GET /especialidades'
---

Retrieves a list of all medical specialties available in the system. This is a public endpoint that does not require authentication.

## Response

<ResponseField name="specialties" type="array">
  Array of specialty objects
  
  <ResponseField name="id" type="number">
    The unique identifier for the specialty
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    The name of the medical specialty
  </ResponseField>
  
  <ResponseField name="descripcion" type="string">
    A description of the specialty (optional)
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    Timestamp when the specialty was created
  </ResponseField>
</ResponseField>

## Example Request

```bash
GET /especialidades
```
