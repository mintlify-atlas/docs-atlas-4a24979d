---
title: "Get Feature Information"
api: "POST /chatbot/feature"
---

Retrieve detailed information about a specific feature in the SaludMap system using the AI chatbot.

## Request Body

<ParamField body="featureId" type="string" required>
  Identifier of the feature to get information about
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Container for the response data
  
  <ResponseField name="featureId" type="string">
    The requested feature identifier
  </ResponseField>
  
  <ResponseField name="response" type="string">
    AI-generated information about the feature
  </ResponseField>
  
  <ResponseField name="timestamp" type="string">
    ISO 8601 timestamp of when the response was generated
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X POST https://api.example.com/chatbot/feature \
  -H "Content-Type: application/json" \
  -d '{
    "featureId": "mapa-interactivo"
  }'
```

## Example Response

```json
{
  "success": true,
  "data": {
    "featureId": "mapa-interactivo",
    "response": "El Mapa Interactivo es una funcionalidad que permite visualizar centros de salud y veterinarias cercanas en un mapa dinámico. Los usuarios pueden buscar por ubicación, filtrar por especialidad, y ver información detallada de cada centro.",
    "timestamp": "2026-03-09T10:30:45.123Z"
  }
}
```

## Error Responses

<ResponseField name="statusCode" type="number">
  HTTP status code
</ResponseField>

<ResponseField name="message" type="string">
  Error message description
</ResponseField>

### 500 Internal Server Error

Returned when there is an error generating the feature information.

```json
{
  "statusCode": 500,
  "message": "Error al obtener información de la funcionalidad"
}
```
