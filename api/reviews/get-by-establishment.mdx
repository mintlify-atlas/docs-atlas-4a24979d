---
title: 'Get Reviews by Establishment'
api: 'GET /resenias/establecimiento/:id'
---

Retrieves all reviews for a specific healthcare establishment. This is a public endpoint that does not require authentication.

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the healthcare establishment
</ParamField>

## Response

<ResponseField name="reviews" type="array">
  Array of review objects for the establishment
  
  <ResponseField name="id" type="number">
    The unique identifier for the review
  </ResponseField>
  
  <ResponseField name="usuarioId" type="number">
    The ID of the user who created the review
  </ResponseField>
  
  <ResponseField name="turnoId" type="number">
    The ID of the appointment
  </ResponseField>
  
  <ResponseField name="establecimientoId" type="number">
    The ID of the healthcare establishment
  </ResponseField>
  
  <ResponseField name="puntuacion" type="number">
    Rating score (1-5)
  </ResponseField>
  
  <ResponseField name="comentario" type="string">
    The review comment
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    Timestamp when the review was created
  </ResponseField>
  
  <ResponseField name="usuario" type="object">
    User information (typically anonymized)
    
    <ResponseField name="id" type="number">
      User ID
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      User first name
    </ResponseField>
    
    <ResponseField name="apellido" type="string">
      User last name
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="average" type="number">
  Average rating across all reviews
</ResponseField>

<ResponseField name="total" type="number">
  Total number of reviews
</ResponseField>

## Example Request

```bash
GET /resenias/establecimiento/45
```

## Error Responses

### Establishment Not Found

```json
{
  "statusCode": 404,
  "message": "Establishment not found",
  "error": "Not Found"
}
```

### Invalid ID

```json
{
  "statusCode": 400,
  "message": "Invalid establishment ID",
  "error": "Bad Request"
}
```

## Use Cases

- Display reviews on establishment detail pages
- Calculate and show average ratings
- Help users make informed decisions about healthcare providers
- Provide social proof for healthcare establishments
